Домашнее задание к 19-му уроку.

Напишите множественное условие для действий при различных сигналах светофора.
Если сигнал красный, то надо стоять, иначе, если желтый, то надо приготовиться, а
иначе, можно идти.

var redLight = 0,
    yellowLight = 0,
    greenLight = 1;

if (redLight == true) {
  document.write('Attention, red light, don\'t go!');
} else if (yellowLight == true) {
  document.write('Attention, yellow light, get ready!');
} else {
  document.write('Attention, green light, you can go!');
}
