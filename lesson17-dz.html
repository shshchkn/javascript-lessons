Домашнее задание к 17-му уроку.

Возьмите массив студентов из 15-го урока:

Напишите условие, которое проверяет, если количество элементов в массиве больше или равно 3,
то выводится сообщение о том, что это большой массив, в котором как минимум 3 элемента.
А иначе, выведите на экран сообщение о том, что это маленький массив, в котором менее 3-х элементов.

var students = ['Григорий Щепкин', 'Тамара Андреева', 'Василий Уткин', 'Петр Смирнов', 'Алексей Петров'];

if (students.length >= 3) {
document.write('Это большой массив, в котором как минимум 3 элемента.');
} else {
document.write('Это маленький массив, в котором менее 3-х элементов.');
}
